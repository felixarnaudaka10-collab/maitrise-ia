Le hack secret

🧠 Laisse Claude installer les meilleurs skills

Au lieu de chercher des heures, tu donnes à Claude la page officielle des skills et tu le laisses choisir pour toi.

1

Copie l'URL du repo officiel Anthropic

repo officiel
https://github.com/anthropics/skills
2

Colle-la dans Claude Code avec ce prompt

Prompt
Voici le repo officiel des skills Claude :
https://github.com/anthropics/skills

En te basant sur ce que tu sais de moi et de mon
activité, trouve les skills les plus utiles pour moi,
explique pourquoi, et installe-les.
💡 Astuce : plus Claude te connaît (Projet avec ton contexte, CLAUDE.md…), plus ses recommandations sont pertinentes. Tu peux aussi parcourir des listes comme awesome-claude-skills.
Rappel

⚙️ Installer un skill, vite fait

A

Le plus simple

Dans Claude Code : donne le lien GitHub du skill et demande « installe ce skill ». Il s'en occupe.

B

À la main

Récupère le dossier du skill (avec son SKILL.md) et place-le dans ton dossier skills/. Relance Claude.
