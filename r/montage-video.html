🎬 Guide complet — Montage vidéo avec Claude

Ce guide couvre tout : ce que Claude peut faire avec Remotion, comment l'installer, les deux workflows principaux, et comment prompt Claude pour obtenir des résultats précis.

TL;DR — Ce qu'il faut retenir

  • Remotion est un framework qui crée des vidéos en code React. Tu décris, Claude écrit le code, Remotion génère le MP4.
  • Le Remotion Agent Skill apprend à Claude comment écrire du code Remotion correctement. Tu l'installes une fois, ça change tout.
  • Deux workflows : incrustations et animations sur une vidéo propre, ou découpe assistée via transcription Whisper.
  • Ce qui marche très bien : sous-titres animés mot par mot, motion graphics, lower thirds, intros de logo.
  • Ce qui ne marche pas : suppression automatique des mots parasites frame-perfect. C'est une limite technique.
  • Ton fichier source doit être en H.264 MP4. Si tu filmes sur Mac ou iPhone, il faudra convertir d'abord.
  • Tout se rend en local. Aucun service cloud. Aucun frais à la minute.
PARTIE 1
Qu'est-ce que Remotion et pourquoi ça change tout
Les éditeurs classiques (Premiere, DaVinci) fonctionnent avec une timeline visuelle — intuitif mais 100% manuel. Remotion aborde la vidéo différemment : tout est du code React. Tu écris ce qui doit apparaître à chaque image, Remotion génère le MP4.

L'avantage : si tu peux décrire une vidéo en français, et que Claude peut écrire le code React à ta place, tu contournes complètement la courbe d'apprentissage. C'est exactement ce que permet le Remotion Agent Skill.
Fondamentaux
PARTIE 2
Ce que l'IA peut et ne peut pas faire
✓ Ce qui marche : sous-titres animés synchronisés, découpe de segments par timestamps, motion graphics et incrustations (lower thirds, titres animés, intros de logo).

✗ Ce qui ne marche pas : suppression automatique des mots parasites frame-perfect. L'IA traite la vidéo par images échantillonnées — ce n'est pas un problème de prompting, c'est une limite technique fondamentale.
Important à savoir
PARTIE 3
Installation — Les deux workflows
Scénario A (recommandé) — Incrustations et Animations : Tu as une vidéo propre et tu veux ajouter des sous-titres, motion graphics ou effets par-dessus.

Scénario B — Découpe assistée par IA : Tu as des rushs bruts, tu veux utiliser Whisper pour transcrire l'audio, puis Claude pour identifier les segments à garder.

Dans les deux cas : installe Node.js, crée un projet Remotion avec npx create-video@latest, puis installe le skill avec npx skills add remotion-dev/skills.
Setup
PARTIE 4
Ajouter des sous-titres et motion graphics
Exemple de prompt qui fonctionne :

"En utilisant le skill remotion-best-practices, prends la vidéo dans public/output.mp4 et ajoute des sous-titres animés mot par mot en police Inter blanche, 36px, centrés en bas de l'image, avec le mot actuel en jaune. Synchronise le timing avec l'audio."
Règle d'or : plus tu es précis sur la police, la taille, la couleur et la position — meilleur est le résultat.
Pratique
PARTIE 5
Découpe assistée avec Whisper + Claude
Installe Whisper : npx remotion whisper install
Transcris ta vidéo : npx remotion whisper transcribe public/output.mp4

Donne ensuite le JSON de transcription à Claude et demande-lui de générer une liste des segments à garder/couper. Remotion rend uniquement les segments "keep" avec un fondu enchaîné de 3 images entre chacun.

Note : les points de coupe ne sont pas toujours parfaits — prévois de les vérifier manuellement dans le JSON.
Avancé
PARTIE 6
Problèmes courants et solutions
Vidéo ne se lit pas ou sous-titres désynchronisés
Convertis en H.264 : npx remotion ffmpeg -i source.mov -c:v libx264 -crf 23 output.mp4
Erreurs dans Remotion Studio
Mets à jour le skill : npx remotion skills update
Transcription Whisper trop lente
Utilise le modèle medium.en au lieu de large.
Dois-je connaître React ou JavaScript ?
Non. Claude écrit tout le code. Savoir lire un peu de code aide quand il y a un problème.
