Partie 1 · Les outils

🔌 Les 2 MCP officiels

Un MCP, c'est le standard ouvert créé par Anthropic pour brancher Claude sur des données externes. Ces deux-là sont officiels, maintenus par les éditeurs eux-mêmes — pas des projets tiers abandonnés.

🏛️ Data Gouv 100% gratuit

La plateforme officielle de l'open data français. Lecture seule, aucune clé API.

  • 40 000+ datasets publics (INSEE, ministères, collectivités)
  • Population, emploi, revenus, démographie
  • Parcelles cadastrales & DVF (prix immobilier)
  • Aides publiques, subventions, marchés publics
  • Requêtes SQL-like directement sur les ressources
Serveur officiel : https://mcp.data.gouv.fr/mcp · github.com/datagouv/datagouv-mcp

📊 Pappers Gratuit puis ~25€/mois

La base française de référence sur les entreprises. 35 outils, données fraîches et tracées.

  • 11 433 321 entreprises françaises (SIREN actifs)
  • Dirigeants, bénéficiaires effectifs, dates de naissance
  • Bilans : CA, effectif, résultat net, dettes, scoring
  • BODACC + procédures collectives (redressement / liquidation)
  • Base immobilière DVF + décisions de justice
  • Outil de sirétisation (enrichis ta propre base)
Serveur officiel : https://mcp.pappers.fr/{ta-clé} · pappers.fr/mcp
💡 Les deux se complètent. Data Gouv pour l'open data macro et l'immobilier (gratuit, illimité). Pappers pour l'analyse entreprise fine (dirigeants, bilans, risques). Claude les croise automatiquement dans une même conversation.
Partie 2 · Installation

⚙️ L'installation en 5 minutes

Deux méthodes. La plus simple = les connecteurs Claude Desktop (2 clics). La plus robuste = le fichier de config.

✅ Option A — Connecteurs (le plus simple)
1

Ouvre Claude Desktop → Paramètres → Connecteurs

Clique sur « Ajouter un connecteur personnalisé ».

2

Colle l'URL de Data Gouv (aucune clé)

Data Gouv — URL
https://mcp.data.gouv.fr/mcp
3

Ajoute Pappers avec ta clé API

Remplace {ta-clé} par ta clé API Pappers (étape juste en dessous).

Pappers — URL
https://mcp.pappers.fr/{ta-clé}
📁 Option B — Fichier de config (avancé)
1

Localise claude_desktop_config.json

Mac : ~/Library/Application Support/Claude/claude_desktop_config.json
Windows : %APPDATA%\Claude\claude_desktop_config.json
Linux : ~/.config/Claude/claude_desktop_config.json
S'il n'existe pas, crée-le.

2

Colle cette config (la vraie, vérifiée)

claude_desktop_config.json
{
  "mcpServers": {
    "datagouv": {
      "command": "npx",
      "args": ["-y", "mcp-remote",
               "https://mcp.data.gouv.fr/mcp"]
    },
    "pappers": {
      "type": "url",
      "url": "https://mcp.pappers.fr/TA_CLE_API"
    }
  }
}
⚠️ Windows : si le serveur apparaît mais ne se connecte jamais, ajoute "isUsingBuiltInNodeForMcp": false à la racine du fichier, puis relance.
🔑 Récupérer ta clé API Pappers
→

Crée un compte gratuit sur pappers.fr

Espace client → API / Clés API → Créer une clé. Copie-la et colle-la à la place de {ta-clé} / TA_CLE_API. Le plan gratuit offre des crédits pour tester ; au-delà ~25€/mois (10× moins cher qu'Apollo).

🔄 Relance Claude Desktop
→

Ferme complètement Claude (Cmd+Q sur Mac) puis rouvre

Les 2 MCP apparaissent dans le menu en bas (icône prise 🔌).

✅ Test rapide : ouvre un chat et tape « Combien de SAS ont été créées après 2023 dans le 75001 ? ». Si Claude répond avec un chiffre exact en citant Data Gouv / Pappers comme source, c'est bon.
Partie 3 · Cas d'usage

🎯 Ce que tu peux faire en langage naturel

Plus besoin de coder, de requêter une API ou de payer 400–500€ une prestation. Tu demandes, Claude répond, sourcé.

🎯 Qualifier un prospect

« Donne-moi les agences de comm créées à Bordeaux en avril 2026 » → liste avec SIREN, fondateur, CA, en 3 secondes.

🏠 Décider en immobilier

« Les transactions immo à Paris 6e ? » → prix moyen au m², 10 dernières ventes, le tout sourcé DVF.

⚠️ Détecter un client à risque

« Check le SIRET de mes 5 derniers clients sur le BODACC » → repère un redressement ou une liquidation avant tes concurrents et récupère ta créance.

📈 Sourcer en masse

« Liste les PME industrielles en Bretagne avec CA > 5M€ et exporte en CSV » → ta base de prospection prête à attaquer.

Partie 4 · Les prompts

💬 5 prompts prêts à copier-coller

Enchaîne-les dans une même conversation : Claude garde le contexte et croise Data Gouv + Pappers automatiquement.

1Chasser des prospects ciblés
Prompt 1
Cherche les SAS en NAF 70.21Z, 73.11Z et 82.99Z
créées après 2023, localisées à Paris, 92, 69 ou 33,
avec un effectif > 2. Enrichis avec dirigeant principal
et chiffres clés. Classe par croissance du CA décroissante.
Sors un tableau : Nom | SIREN | Ville | Effectif | CA | Dirigeant.
2Détecter les pépites en création
Prompt 2
Liste les SAS créées dans les 6 derniers mois en NAF
62.02A (conseil IT) avec un capital social > 10 000 €
et un objet social mentionnant "IA", "automatisation"
ou "agent". Sors les 50 plus prometteuses avec une
raison pour chacune.
3Enrichissement contact en masse
Prompt 3
Voici 50 SIREN : [colle-les séparés par des virgules].
Pour chacun, récupère via Pappers le dirigeant principal
(nom + fonction), l'email et le téléphone RCS si
disponibles, et le site web déclaré. Sors un CSV propre.
4Analyse immobilière d'un quartier
Prompt 4
Via DVF, donne-moi le prix moyen au m² à Paris 6e
arrondissement et les 10 dernières transactions
(adresse, surface, nb de pièces, prix). Ajoute la
tendance sur 2 ans. Cite les sources.
5Veille risque clients (BODACC)
Prompt 5
Voici les SIRET de mes clients : [colle-les].
Vérifie sur le BODACC s'il y a une procédure collective
en cours (sauvegarde, redressement, liquidation).
Résume en bullet points avec le statut et la date de
chaque procédure, et les liens sources.
