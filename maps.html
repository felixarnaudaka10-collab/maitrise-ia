Sans Claude

La version manuelle

Si tu préfères passer par le terminal directement, voici la commande Docker officielle.

mkdir -p gmaps-output echo "dentistes Paris" > queries.txt docker run \ -v gmaps-playwright-cache:/opt \ -v "$PWD/queries.txt:/queries.txt:ro" \ -v "$PWD/gmaps-output:/out" \ gosom/google-maps-scraper \ -input /queries.txt \ -results /out/results.csv \ -depth 1 -email \ -exit-on-inactivity 3m
Le flag -email visite chaque site web pour en extraire les emails. C'est plus lent, mais c'est là que se trouve la vraie valeur.
