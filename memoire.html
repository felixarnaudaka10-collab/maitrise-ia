Installation

⚙️ En 2 minutes

Pré-requis : avoir Claude Code installé. Une seule commande.

1

Installe le plugin

Terminal
npx claude-mem install

⚠️ Utilise bien npx claude-mem install (pas npm install -g qui n'installe que la lib, sans brancher le plugin).

2

Relance Claude Code

Le plugin se branche automatiquement. À partir de là, Claude capture et réutilise le contexte tout seul.

3

Vérifie

Lance une session, ferme-la, rouvre : Claude se souvient de ce que vous avez fait. C'est bon.

📂 Tout le détail (commandes /plugin, options, sync multi-machines) est dans le repo officiel : github.com/thedotmack/claude-mem
* Honnêteté : le "10× / jusqu'à 95%" dépend de ton usage (gros gains sur les sessions répétitives, plus modeste sinon). Mais le gain est réel.
