Script Reel Instagram viral
Créer un script complet pour un Reel de 30-60 secondes
✅ OFFERT
🎁 Prompt complet offert — la qualité de toute la bibliothèque
Tu es un scriptwriter spécialisé en Reels Instagram viraux. Mon activité : [décris ton business] Mon audience : [qui te suit] Le sujet du Reel : [ton idée ou message principal] Écris un script complet de Reel de 30 à 60 secondes : 1. HOOK (0-3 sec) : 3 variantes de première phrase qui stoppent le scroll (pattern interrupt, question choc, affirmation contre-intuitive) 2. CORPS (3-45 sec) : le contenu découpé plan par plan, avec les textes à l'écran (overlays) pour chaque plan 3. CTA (45-60 sec) : une seule action claire à faire 4. CAPTION : légende avec accroche + 5 hashtags de niche Règles : phrases de 8 mots max, une idée par plan, écris comme on parle.
💡 Tip d'expert : tourne les 3 hooks et publie celui qui te met le moins à l'aise. C'est presque toujours lui qui performe le mieux.

41
Créer un agent IA conversationnel
Construire le system prompt d'un agent Claude sur mesure
✅ OFFERT
🎁 Prompt complet offert — la qualité de toute la bibliothèque
Tu es un architecte d'agents IA. Je veux créer un agent Claude pour : [décris la mission de l'agent] Ses utilisateurs : [qui va lui parler] Ce qu'il doit toujours faire : [règles obligatoires] Ce qu'il ne doit jamais faire : [limites] Construis le system prompt complet de cet agent : 1. RÔLE : identité et expertise de l'agent en 2 phrases 2. CONTEXTE : ce qu'il sait de l'entreprise et des utilisateurs 3. INSTRUCTIONS : son process étape par étape pour chaque demande 4. RÈGLES : les obligations et interdictions formelles 5. TON : comment il s'exprime, avec 2 exemples de réponse type 6. CAS LIMITES : quoi faire quand il ne sait pas répondre Puis pose-moi 3 questions pour affiner avant la version finale.
💡 Tip d'expert : la section 6 fait la différence entre un agent pro et un gadget. Un agent qui sait dire "je ne sais pas" inspire confiance et évite les erreurs coûteuses.
