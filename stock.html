1

Va sur claude.ai → New Project. Donne-lui un nom : "Analyse Bourse". Les projets conservent le contexte entre les sessions.
2
Colle le prompt Smart Money ci-dessous
Utilise le prompt exact de la section suivante. Il donne à Claude son rôle d'analyste et la structure d'output attendue.
3
Copie-colle les données sources
Whale Alert (Twitter/X), quiverquant.com pour les trades du Congrès, SEC EDGAR pour les 13F de BlackRock. Claude fait le reste.
4
Lis le signal global et décide
Claude t'affiche un verdict clair : HAUSSIER · NEUTRE · BAISSIER avec les raisons. Tu gardes le contrôle, lui fait l'analyse.
