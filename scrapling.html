Partie 2 · Installation

⚙️ Branche-le à Claude en 5 min

Pré-requis : Python 3.10+. On installe Scrapling avec le serveur MCP, puis on l'ajoute à Claude.

1 · Installer Scrapling (+ navigateurs)
Terminal
pip install "scrapling[ai]"
scrapling install
2A · Ajouter à Claude Desktop

Menu ☰ → Réglages → Développeur → Modifier la config. Colle :

claude_desktop_config.json
{
  "mcpServers": {
    "ScraplingServer": {
      "command": "scrapling",
      "args": ["mcp"]
    }
  }
}
💡 Si Claude ne le détecte pas, mets le chemin complet de scrapling : tape which scrapling (Mac) ou where scrapling (Windows) et remplace "scrapling" par le chemin retourné.
2B · Ou via Claude Code (1 commande)
Terminal
claude mcp add ScraplingServer "$(which scrapling)" mcp
3 · Relance Claude

Quitte complètement Claude puis rouvre. Tu verras l'indicateur MCP (🔧) en bas du chat → ScraplingServer.

✅ Test : « Récupère le contenu principal de https://example.com en markdown. » Si Claude utilise l'outil Scrapling et te renvoie le contenu, c'est bon.
Partie 3 · Prompts

💬 Prompts prêts à copier-coller

Astuce : précise toujours quel mode utiliser (« requêtes normales » pour aller vite, navigateur si le site est dynamique).

1Extraire une page en markdown propre
Prompt 1
Avec des requêtes normales, récupère le contenu
principal de [URL] et convertis-le en markdown
propre (main_content_only).
2Extraire des éléments précis (CSS)
Prompt 2
Récupère tous les titres de produits de [URL] avec
le sélecteur CSS '.product-title'. Si tu ne connais
pas le bon sélecteur, essaie plusieurs combinaisons
jusqu'à trouver le bon.
3Collecter prix/produits en masse
Prompt 3
Fais un bulk fetch sur ces URLs publiques et renvoie
un tableau Nom | Prix | Description :
- [URL 1]
- [URL 2]
- [URL 3]
4Crawl d'une catégorie
Prompt 4
Va sur cette URL de catégorie, extrais toutes les URLs
produits avec le sélecteur 'a', puis fetch les 3
premières pages en parallèle et sors le prix + les
détails de chaque produit. Markdown. URL : [URL]
⚖️ Scrape responsable. Vérifie le robots.txt du site, respecte ses conditions d'utilisation et les limites de débit, et reste sur des données publiques. C'est aussi ce que recommande la doc officielle de Scrapling.
